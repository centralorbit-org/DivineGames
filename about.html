About DivineGames

What this project is, how it works, and the important stuff.

What is DivineGames?

DivineGames is a simple web launcher interface that helps you load a game by selecting or entering a game slug (the part of the game URL that identifies the game).

The goal is a clean, fast UI: search, categories, and a straightforward “type slug → play” workflow.

How does it work?

DivineGames reads the ?game= parameter in the URL (for example: ?game=crazy-flips-3d) and then loads the game using your existing JavaScript logic.

If you don’t provide a slug, it stays in “launcher mode” where you can browse/search slugs and start a game.
